## 학습 내용
1. 레이아웃 재사용성 높이기 위해 리팩토링 완료
    - 폴더 구성하고 app.vue에 추가하여 새로운 페이지 만들때 일일히 추가하는 번거로움 사라짐

2. 다크모드에서 글자가 잘 안보이는 현상 해결
    - 다크모드시 배경색은 white임에도 불구하고 텍스트를 gray 혹은 white로 지정하여 잘안보이는 현상
    - 다크모드는 일단 배제 후 차후 다크모드 추가 예정

3. 모바일 반응형 문제
    - 웹과 모바일 환경을 맞추기 위해 464px로 고정했는데 이때 모바일에선 더 작게 나오는 현상
    - 문제점: 모바일은 기본적으로 웹 페이지를 980px 데스크탑 기준으로 축소해서 보여줌
    - <meta name="viewport" content="width=device-width, initial-scale=1.0" /> html 파일에 추가하여 모바일에서 접속해도 억지로 화면을 줄여서 보여주지 않음
  
4. Oaut(naver, kakao) 로그인 user정보와 토큰 받아오기
```js
calStorage.setItem('access_token', res.data.access)
localStorage.setItem('refresh_token', res.data.refresh)
localStorage.setItem('user', JSON.stringify(res.data.user)) 
```
- 이건 국룰로 받아와야 되는걸로

5. 네이버 지도 검색 기능
- kakao window.daum.Postcode기능 사용하여 파업창 띄운뒤 주소입력
- 해당 주소를 naver geocode API 활용하여 위도 경도로 변경
- 위도, 경도, 주소를 벡엔드에 전송
