## 학습 내용
- Not-found
```js
 {
        path: '/:pathMatch(.*)*',
        name: 'NotFound',
        component: NotAuthenticated,
  },
```
해당 경로로 not-found 전용 page를 만들어서 이동

- 로그인 인증 시에만 main 화면 접속
    - router.beforeach 기능 사용
    - 로그인 인증 access_token이 없을때 main 화면 접속 불가
    - django에서 친절하게 access_token, refresh_token, user(data)를 다 넘겨줌
    - index.js에서
    ```js
        router.beforeEach((to, from, next) => {
        const token = localStorage.getItem('access_token')

        if (to.meta.requiresAuth && !token) {
            return next('/notauthenticated')      // 로그인 안됐으면 로그인 창으로 이동
        }

        next() // 통과
        })

    ```
    확인하고 통과 혹은 Not-found 화면에 보냄
